# Ejecutar En Desarollo
1. Clonar repositorio
2. Ejecutar:
```
npm install
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```
4. Levantar base de datos
```
docker-compose up -d

```
5. Reconstruir la semilla
http://localhost:3000/api/v2/seed?size=40
